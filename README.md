Gratitude
=========

This is the source code of an application to reinforce your gratitude attitude.

## Project setup

`vagrant up` and access the application at `http://gratitude.lo/app_dev.php/`.

## Deployment

### Using Heroku

To configure the platform (to do once), take a look at `app.json` and configure Heroku accordingly.

Then, simply run `git push heroku master`.
